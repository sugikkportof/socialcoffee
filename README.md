# ☕ Singaraja Social Coffee — Premium Landing Page

![Singaraja Social Coffee Banner](https://images.unsplash.com/photo-1497935586351-b67a49e012bf?auto=format&fit=crop&w=1200&q=80)

## 📌 Tentang Proyek
Website *landing page* premium yang dirancang khusus untuk **Singaraja Social Coffee**, sebuah *specialty coffee shop* dan *event space* di Singaraja, Bali. Website ini berfokus pada pengalaman visual yang mewah (UI/UX), performa pemuatan yang cepat, dan sistem konversi reservasi langsung melalui WhatsApp.

🔗 **Live Demo:** [Lihat Website Disini]([LINK-VERCEL-LU-NANTI-DISINI])

## ✨ Fitur Unggulan
* **Modern Warm Clean UI:** Desain antarmuka eksklusif menggunakan palet warna *cream, rich brown,* dan aksen *gold*.
* **Glassmorphism Navbar:** Navigasi atas yang transparan dan *blur* dengan efek garis bawah animasi (*animated underline*) saat disentuh.
* **Micro-interactions & Animations:** Efek *hover bouncy* pada kartu fitur dan animasi *scroll* yang halus menggunakan AOS.js.
* **Interactive Digital Menu:** Slider menu minuman unggulan (Swiper.js) dan sistem *grid* menu lengkap dengan fitur filter kategori (*Smooth Fade*).
* **Filterable Masonry Gallery:** Galeri foto ruang *indoor, outdoor,* dan *event space* yang interaktif dilengkapi dengan fitur *Lightbox* (Pop-up gambar).
* **Smart WhatsApp Reservation Form:** Formulir pemesanan *event space* dinamis yang otomatis merangkum data pengguna menjadi pesan WhatsApp yang rapi ke pihak manajemen kafe.

## 🛠️ Teknologi yang Digunakan
* **HTML5** (Semantik dan SEO Friendly)
* **Tailwind CSS** (Styling via CDN untuk *rapid UI development*)
* **Vanilla JavaScript** (Logika DOM, Filter Menu, dan Integrasi WhatsApp)
* **AOS.js** (Animasi *Scroll*)
* **Swiper.js** (*Touch Slider* responsif)

## 🚀 Cara Menjalankan Secara Lokal
1. *Clone repository* ini ke komputer Anda.
2. Buka file `index.html` menggunakan *browser* pilihan Anda (Chrome/Safari/Firefox).
3. Pastikan koneksi internet aktif karena proyek ini memuat Tailwind CSS dan *library* lainnya melalui CDN.
